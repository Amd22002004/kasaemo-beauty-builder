

# Сайт студии-школы массажа KASAEMO

## Дизайн
- Фуксия (#FF2D95) как основной акцент, светло-розовые фоны, белый и серый для текста
- Заголовки блоков крупные, uppercase
- Плавные анимации при скролле и hover
- Адаптивная верстка для всех устройств
- Фото команды используется в секции «О нас»

## Страницы

### Главная (/)
- **Hero-баннер** в розово-фуксиевой гамме с УТП «Мы искренне любим ваше лицо и тело — комплексный подход к красоте и здоровью»
- CTA кнопки: «Записаться» (ссылка на YCLIENTS), «Прайс», «Наши специалисты»
- Секция «О нас» — миссия, часы работы (09:00–22:00), кликабельный телефон, ссылка на VK, фото команды
- Краткий блок популярных услуг (карточки с ценами)
- Блок отзывов (загружаются из БД)
- FAQ / Противопоказания

### Услуги и прайс (/uslugi)
- Карточки всех 19 услуг с названием и ценой/диапазоном
- Фильтр по категориям: Лицо / Тело / Курсы / Сертификаты
- Кнопка «Записаться» на каждой карточке → YCLIENTS

### Специалисты (/specialisty)
- Грид 3×3 (9 карточек): фото-плейсхолдер, ФИО, специализация, краткое описание
- Кнопка «Записаться к специалисту» → YCLIENTS
- Данные загружаются из Supabase (редактируемые через админку)

### Курсы и семинары (/kursy)
- Карточки курсов с ценой, описанием
- 4 курса из прайса: базовый курс массажа лица, остеопластика тела, миофасциальная техника, семинар по массажу спины
- Кнопка «Записаться на курс» → YCLIENTS

### Контакты (/kontakty)
- Два адреса: Мельникайте 125Б и Советская 95
- Встроенная карта (OpenStreetMap) с двумя метками
- Кликабельный телефон, часы работы
- Ссылка на VK
- Форма обратной связи (имя, телефон, сообщение → сохранение в Supabase)

### Подарочные сертификаты (/sertifikaty)
- Блок с информацией о сертификатах (1 000–10 000 ₽)
- Кнопка «Записаться» / «Узнать подробности» → YCLIENTS или телефон

## Общие элементы
- **Хедер** — логотип KASAEMO, навигация по страницам, телефон, кнопка «Записаться»
- **Footer** — контакты, оба адреса, ссылка на VK, копирайт, политика конфиденциальности
- **Плавающая кнопка** — VK-чат виджет в углу

## Бэкенд (Lovable Cloud / Supabase)
- **Таблицы**: специалисты, услуги, отзывы, обращения (форма обратной связи)
- **Админ-панель** (/admin) с авторизацией: редактирование специалистов (фото, ФИО, описание), услуг (название, цена, категория), отзывов, обращений
- **Хранилище файлов** для фото специалистов (Supabase Storage)

## SEO
- Title: «KASAEMO — студия-школа массажа лица и тела в Тюмени»
- Meta description с телефоном и часами работы
- H1 на главной: «Студия-школа массажа лица, тела и остеопластики KASAEMO»

## Этапы реализации
1. **Фронтенд**: все страницы со статичным контентом, дизайн, адаптив
2. **Бэкенд**: Supabase таблицы, загрузка данных из БД
3. **Админ-панель**: CRUD для специалистов, услуг, отзывов
4. **Финал**: SEO мета-теги, карта, тестирование
